# Time Series Analysis — Course Notebooks

Companion notebooks for the **Time Series & Forecasting** course on Zylo.

Each notebook is named `{sectionId}-b{blockIndex}.ipynb` and contains the
runnable code from a single section of the course. They are auto-generated
from the course content and opened via the "Run on Colab" button next to
